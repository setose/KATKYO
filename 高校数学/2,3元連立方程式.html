基礎から見直す
連立方程式

「文字が多くて、どこから手をつければいいか分からない…」
今回はパズル感覚で解ける「代入法（入れ替え作戦）」で攻略します。

これだけは覚えて！大原則

連立方程式の目標は
「文字を1つ消す」こと

$x$ と $y$ 、文字が2つあるから解けないんです。だったら、片方の文字の「正体」を暴いて、もう一つの式に当てはめて（入れ替えて）、文字を1種類だけにしてしまえばいい。 これを「代入」と呼びます。

1 2元連立方程式（文字が2つ）

次の問題を一緒に解いてみましょう。文字は $x$ と $y$ の2つです。

\[ \begin{cases} y = 2x - 1 \dots ① \\ x + 2y = 8 \dots ② \end{cases} \]

Step 1：「正体」がわかっている文字を探す

①の式をよく見てください。「$y = \dots$」の形になっていますね。
これは、「$y$ の正体は $2x - 1$ だよ」と教えてくれているんです。

Step 2：もう一つの式と「入れ替える（代入）」

②の式の「$y$」の部分に、暴いた正体「$2x - 1$」をそのまま入れ替えます。
【超重要】入れ替えるときは、必ずカッコ $( \ \ )$ をつけてください！

元の②の式： $x + 2\textcolor{blue}{y} = 8$

入れ替えた後： $x + 2\textcolor{blue}{(2x - 1)} = 8$

どうですか？式の中から $y$ が完全に消滅しました！

Step 3：文字が1つになった式を解く

あとは中学生の最初で習った普通の方程式です。カッコを外して解きましょう。

\[ x + 4x - 2 = 8 \] \[ 5x - 2 = 8 \] \[ 5x = 10 \] \[ x = 2 \]

これで片方の答えが出ました。

Step 4：元の式に戻して、もう1つも出す

分かった $x = 2$ を、計算しやすそうな最初の①の式に入れます。

\[ y = 2 \times 2 - 1 \] \[ y = 4 - 1 \] \[ y = 3 \]

答え： $x = 2, \ y = 3$

✍️ 練習問題（文字が2つ）

最初から「$y = \dots$」の形になっていない問題です。まずは自分で、どちらかの式を「$x = \dots$」または「$y = \dots$」の形（正体を暴く形）に変形させるところから始めましょう。

\[ \begin{cases} x - 3y = 4 \dots ① \\ 2x + y = 1 \dots ② \end{cases} \]
解答と解説を見る

解説：

まず「正体」を作ります。①の式は $x$ に数字がついていないので、変形しやすそうです。

①を変形： $x = 3y + 4 \dots ①'$

これで「$x$ の正体は $3y + 4$」だと分かりました。これを②の式の $x$ の部分に入れ替えます。（必ずカッコをつけて！）

\[ 2(3y + 4) + y = 1 \] \[ 6y + 8 + y = 1 \] \[ 7y = 1 - 8 \] \[ 7y = -7 \] \[ y = -1 \]

$y = -1$ が出ました。これを、さっき自分で作った正体の式（①'）に入れます。

$x = 3 \times (-1) + 4$
$x = -3 + 4$
$x = 1$

答え： $x = 1, \ y = -1$

2 3元連立方程式（文字が3つ）

文字が $x, y, z$ と3つに増えました。でもやることは同じです。
「1つの文字の正体を暴いて、残り全部の式にばらまく」。これで文字が1つ消滅します。

\[ \begin{cases} x = y + 1 \dots ① \\ x + 2y + z = 8 \dots ② \\ 3x - y - z = 1 \dots ③ \end{cases} \]

Step 1：「正体」がわかっている文字を見つける

①の式から、「$x$ の正体は $y + 1$」だと分かっています。

Step 2：残りすべての式からターゲット($x$)を消し去る

②と③の式にある $x$ を、すべて正体の $(y + 1)$ に入れ替えて、$x$ をこの世から追い出します。

②の式に入れ替える：

$(y + 1) + 2y + z = 8$

（整理すると）

$3y + z = 7 \dots ④$

③の式に入れ替える：

$3(y + 1) - y - z = 1$

$3y + 3 - y - z = 1$

（整理すると）

$2y - z = -2 \dots ⑤$

Step 3：見慣れた「2文字の連立方程式」の完成！

新しくできた④と⑤の式を見てください。

\[ \begin{cases} 3y + z = 7 \dots ④ \\ 2y - z = -2 \dots ⑤ \end{cases} \]

これ、第1章でやったものと全く同じ（文字が2つの問題）ですよね！
ここまで来れば勝ったも同然です。これも代入法で解いてみましょう。
④を「$z = \dots$」の形にします。
$z = -3y + 7$
これを⑤の $z$ に入れ替えます（カッコを忘れずに！）。
$2y - (-3y + 7) = -2$
$2y + 3y - 7 = -2$
$5y = 5$
$y = 1$

出た $y = 1$ を使って残りを求めます。
$z = -3 \times 1 + 7 = 4$
最後に、一番最初の①の式に $y=1$ を入れて、
$x = 1 + 1 = 2$

答え： $x = 2, \ y = 1, \ z = 4$

✍️ 練習問題（文字が3つ）

自分で「正体」を作るところからスタートです。数字がついていない文字（今回は①の $x$ や $y$）を選ぶと変形が簡単です。少し計算が長いので、余白に大きく丁寧に書くのがミスを減らすコツです。

\[ \begin{cases} x - y = 2 \dots ① \\ 2x + y - z = -1 \dots ② \\ x - 3y + 2z = 8 \dots ③ \end{cases} \]
解答と解説を見る

解説：

まず、①の式を変形して「正体」を作ります。

①を変形： $x = y + 2 \dots ①'$

1. ②と③の式から $x$ を消す

②と③の $x$ の部分に、$(y + 2)$ を入れ替えます。

【②に入れ替える】

\[ 2(y + 2) + y - z = -1 \] \[ 2y + 4 + y - z = -1 \] \[ 3y - z = -5 \dots ④ \]

【③に入れ替える】

\[ (y + 2) - 3y + 2z = 8 \] \[ -2y + 2z = 6 \] \[ -y + z = 3 \dots ⑤ \]

2. ④と⑤で2元の連立方程式を解く

④と⑤を使って、$y$ と $z$ を求めます。

\[ \begin{cases} 3y - z = -5 \dots ④ \\ -y + z = 3 \dots ⑤ \end{cases} \]

これも代入で解きましょう。⑤を変形して $z = y + 3$ にします。
これを④の $z$ に入れ替えます。

$3y - (y + 3) = -5$
$3y - y - 3 = -5$
$2y = -2$
$y = -1$

あとは順番に戻していくだけです。
$z = -1 + 3 = 2$
$x = -1 + 2 = 1$

答え： $x = 1, \ y = -1, \ z = 2$

お疲れ様でした。

代入法は「正体を暴いて、入れ替えていくパズル」です。
入れ替えるときの「カッコをつける」ことだけ忘れなければ、確実に文字を減らしていけます。応援しています！
